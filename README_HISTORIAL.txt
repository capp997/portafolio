ACTUALIZACIÓN HISTORIAL DEL PORTAFOLIO

IMPORTANTE:
No borra tus activos ni tus datos actuales.

INSTALACIÓN:

1. Copia estas carpetas dentro de:
C:\xampp\htdocs\portafolio-dashboard-v4\

- api
- pages
- assets

2. Abre phpMyAdmin:
http://localhost/phpmyadmin

3. Selecciona la base de datos:
portafolio_db

4. Ve a SQL y ejecuta el contenido de:
install_history.sql

5. Abre:
http://localhost/portafolio-dashboard-v4/pages/historial.php

USO:

- Presiona “Guardar snapshot” para guardar el valor actual del portafolio.
- Hazlo 1 vez al día o en tu día de rebalanceo.
- Con varios registros, la gráfica histórica empezará a mostrar crecimiento.

RECOMENDACIÓN:
Después agregaremos un botón en el menú principal del dashboard para entrar directo a Historial.

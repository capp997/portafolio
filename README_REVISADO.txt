PROYECTO REVISADO Y CORREGIDO

Cambios aplicados:
- Menú uniforme en Dashboard y páginas principales.
- Se agregaron Dividend Tracker, Centro Alertas e Historial Pro al menú.
- Sidebar con botones funcionales: Actualizar precios, Snapshot, Escanear alertas, Dividend Engine y Logout.
- Estilos premium unificados con style_v5.css + unified_pages.css.
- Mobile premium cargado en páginas independientes y páginas con layout.
- index.php redirige automáticamente a index_v5.php.
- Se mantuvieron tus datos y la estructura de base de datos.

Instalación:
1. Reemplaza tu carpeta actual portafolio-dashboard-v4 por esta versión.
2. No importes database.sql si ya tienes tus datos.
3. Abre http://localhost/portafolio-dashboard-v4/

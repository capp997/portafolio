PORTAFOLIO DASHBOARD V3

INSTALACIÓN:
1. Copia la carpeta portafolio-dashboard-v3 a:
   C:\xampp\htdocs\

2. En XAMPP inicia Apache y MySQL.

3. Abre:
   http://localhost/phpmyadmin

4. Importa el archivo database.sql.

5. Abre:
   http://localhost/portafolio-dashboard-v3

PÁGINAS INCLUIDAS:
- Dashboard
- Activos
- Alertas
- Compras
- Dividendos
- Rebalanceo
- Rutinas
- Metas


NUEVO EN V4:
- Puedes agregar nuevos activos desde la página Activos.
- Puedes eliminar activos.
- Cuando agregas una acción nueva, aparece automáticamente en Dashboard, Alertas y Rebalanceo.
